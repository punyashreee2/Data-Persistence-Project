# Data Persistence Project

This Unity project demonstrates data persistence between scenes and sessions.

## Features
- Player name entered in the Start Menu is carried into the Main scene.
- High score is saved and displayed with the player name.
- High score persists between sessions using PlayerPrefs and JSON.

## How to Play
1. Enter your name in the Start Menu.
2. Click Start to launch the game.
3. Break bricks to score points.
4. If you beat the high score, your name and score are saved.
5. Quit and relaunch — the high score persists
